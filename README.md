## Development Setup

Before you do anything, you&rsquo;ll need to have [Git](https://git-scm.com/), [Node](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com/en/) installed.

```shell
git clone https://github.com/cadejscroggins/catz
cd catz
yarn
yarn start
```
